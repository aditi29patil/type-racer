Features

- Strict Syntax Matching: Intercepts keyboard buffers dynamically to filter inputs character-by-character.
- Deterministic Game Loop: Powered by requestAnimationFrame for smooth frame updates and physics layout rendering.
- Power-up Systems: Earn custom mechanics like a De-bugger (Time-dilation/Slow-Mo) or Stack Overflow (Instant clear).
- Dynamic Difficulty: Level-based difficulty scaling that ramps up speed vectors based on real-time compilation scores.

Tech Stack

- Frontend: Semantic HTML5, Custom Cyberpunk CSS Variables
- Logic Engine: Vanilla JavaScript (ES6+)

How to Play

1. Clone or download this repository.
2. Open index.html in any modern web browser.
3. Type the falling code blocks accurately before they breach the system baseline.
4. Press [Space] on an empty input to use the De-bugger. Press [Enter] on an empty input to clear the block with Stack Overflow.
